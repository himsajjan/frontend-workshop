Lists

This is how you make a unordered list

  • Add a "ul" elelment (it stands for unordered list)
  • Add each item in its own "li" (list item) elelment
  • They don't need to be in any particular order (is IS an UN-ordered list)

This is what an ordered list looks like

  1. Notice the new "ol" element wrapping everything
  2. But, the list item (li) elements are the same
  3. Also note how the numbers increment on their own
  4. You should be noticing things in this percise order, because it is an ordered list
